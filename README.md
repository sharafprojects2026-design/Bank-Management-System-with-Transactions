# Advanced Bank Management System - C++

An advanced console-based banking system developed using C++.

## Features
- Show all clients
- Add new clients
- Delete clients
- Update client information
- Find client by account number
- Deposit money
- Withdraw money
- Show total balances
- Save and load data using files

## Technologies Used
- C++
- File Handling
- Structures
- Vectors
- Functions
- Console Application

## Project Description
This project is an upgraded version of the basic banking system.

The system allows full client management and banking transactions through a console interface. All client data is stored in text files and loaded automatically.

Each client contains:
- Account Number
- Pin Code
- Full Name
- Phone Number
- Account Balance

The project also supports transaction operations such as deposits and withdrawals with balance validation.
